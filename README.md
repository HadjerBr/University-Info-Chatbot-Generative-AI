CampusMate is a Retrieval-Augmented Generation (RAG) chatbot designed to answer questions about Harvard College using information extracted from official student handbooks and FAQ resources. It combines vector embeddings and a semantic search backend (Pinecone) with a large language model to provide accurate and context-aware responses.

---

## Features

- Answers student questions based on university handbooks and FAQs  
- Uses RAG architecture with vector embeddings for retrieval  
- Choice of LLM: OpenAI `gpt-4.0-mini`
- Supports contextual conversations with memory (WIP)  
- User-friendly web interface built with Flask + JavaScript  
- Modular data loading and embedding pipelines  
- Easy to extend for other universities or datasets
